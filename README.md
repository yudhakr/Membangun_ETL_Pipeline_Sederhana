# Belajar Fundamental Pemrosesan Data
## ETL Pipeline Sederhana

Pipeline ETL ini dibuat dengan Python untuk mengambil, memproses, dan menyimpan data produk dari website fashion.

## Alur
- **Extract**: Mengambil data dari website
- **Transform**: Membersihkan dan memformat data
- **Load**: Menyimpan ke CSV / Google Sheets
